    HOW TO DEFEND AGAINST TROJANS

    Trojans get their name from the fact that they require your permission to run on your computer, either when you execute the program yourself or when you view a document or picture that runs the program for you.





    DO'S


    INSTALL AND RUN AN INTERNET SECURITY SUITE

    Use your program to run diagnostic scans on a regular basis. It's possible to program the program to run scans at predetermined intervals.

    OPERATING SYSTEM SOFTWARE

    As soon as the software company releases updates, update your operating system's software. Cybercriminals frequently take advantage of security flaws in out-of-date software.

    PASSWORD

    Use a complicated combination of letters, numbers, and symbols to create a unique password for each account.

    FIREWALL

    Firewalls protect your personal information.

    RESTORING

    Make regular backups of your files. If your computer is infected with a Trojan, this will assist you in recovering your data.

    PROTECT E-MAILS

    Scanning an email attachment first can help you remain secure.

    DON'TS


    VISIT UNSAFE WEBSITES

    Some internet security software will warn you if you are going to visit a potentially dangerous website.

    E-MAIL LINKS

    Don't click on a link in an email unless you're sure it's from a trustworthy source. In general, don't open unsolicited emails from unknown senders.

    DOWNLOAD PROGRAMS

    If you don't trust the publisher, don't download or install the software.

    POP-UP

    Don't click on pop-up windows that claim to offer free apps that will help you.

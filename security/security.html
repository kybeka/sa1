    Types of Cyber Security

    What is Cyber Security ?

    Cyber Security, even called electronic information security is used to protect computers, networks, servers, cellphones, data and all types of computers devices from hacking attacks.

    It serves as a security gate in order to make sure that all your personal informations isn't hackable; and with cybercrimes exponentially growing it's fundamental to have cyber security in place for personal, military and business use.

    There are mainly 3 different types of Cyber Security:

    • Hardware Security
    • Software Security
    • Network Security

    Hardware Security

    What is Hardware Security ?

    Hardware Security is vulnerability for all types of physical devices, so rather than the software it's the hardware of a computer system the one affected. it may also refer to the protection of physical sistems from a physical harm like equipment destruction attacks.

    Software Security

    What is Software Security ?

    Software Security is the application of every type of technique that can asses, mitigate and protect software systems from all hacking vulnerabilities. Techniques used to ensure that the softwares continues to function and are safe from any kind of malicious attacks.

    Network Security

    What is Network Security ?

    Network Security is used to protect networks and data from breaches, intrusions and other threats. This term gathers hardware and software solutions and also includes rules and configurations relating to network usability, accessibility, and threat protection for all subsections.

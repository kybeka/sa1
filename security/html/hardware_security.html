    Hardware Security

    Hardware security refers to the physical protection or security of a computer device, as opposed to the software installed on it. Firewalls, proxy servers, and HSMs (hardware security modules) are all typical examples. Hardware Security is used to make computer devices more reliable secure and can also provide an additional layer of protection even to the software.

    It is diferrentiated from Software Security by experts, which state that the key difference between hardware security and software security is that hardware security is implemented using a "non-Turing-machine" (click to see the definition).

    Backup Plan

    What is the purpose of a backup plan?

    The purpose of a backup is to create a copy of data that in case of an accidental mistake or a computer failure, such as a hardware or software failure or a malicious attack by a hacker, with a backup plan the data can be recovered. For more information (click here).

    RAIDs

    What is a RAID?

    RAID means "redundant array of independent disks". It's a data storage virtualisation technology that combines several physical disk components into one or many logical units for data redundancy and performance gain, or both. For more information (click here).

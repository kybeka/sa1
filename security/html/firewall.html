    Firewall

    The firewall is a system that controls all incoming and outgoing network traffic in order to filter all the data and to stop malicious actions. It is first programmed with a predefined set of security rules and allow or discard events. This allows to maintain the security of the network.
    Two general rules application criteria used by firewall are:

    • default-deny: what is explicitly authorized is allowed, the rest is discard
    • default-allow: only what is explicitly forbidden is blocked, the rest is allowed

    The first one provides more security and consequently is the most used.





    Some types of firewall

    Packet filtering firewall, which applies a method of filtering the data packets sent. The data in the network circulates by following different types of protocols, for example TCP/IP. With this system the data is grouped in packets, each one with a header where the relative information is reported, including the IP address of the source and the destination. The firewall analyzes the header and, based on the criteria set, determines whether the packet should be blocked because it is dangerous or if it can pass. It is not a safe solution though, because thanks to the IP spoofing technique an IP can be forged making the firewall activity useless.

    Stateful inspection firewall is a type of firewall similar to the previous one but more complete, because in addition to identifying the packets, traces the connections and their status. With this system, spoofing techniques can be circumvented, but it does not prevent other type of attacks as Dos, for example.

    Hardware firewall is a physical component designed to perform the function of firewall. It contains inside only operating components useful for the activity, so there is only specific dedicated software inside, while the rest is removed.

    Last but not least, as an example, there is a cloud firewall, which is a purely virtual software component, i.e. it does not require the installation of any physical machinery. It provides services related to the network of the internet provider and allows remote management like by a technical team of security experts. In this way, access attempts by third parties can be blocked and thanks to the presence of an expert team the most suitable defensive system are adopted.

    The important thing with firewalls is to choose the most suitable for your case.

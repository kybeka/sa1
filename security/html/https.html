    HTTPS protocol

    What does it mean?

    HyperText Transfer Protocol over Transport Socket Layer / Secure Socket Layer.
    It's a web communication-oriented protocol placed on the application level of the ISO/OSI stack, which uses the HTTP protocol and SSL/TLS for supporting encryption and increasing safety. HTTPS, which is also shortened in HTTP Secure, is specified by RFC 2818 (May 2000). It is used to encrypt and send data between the browser and the connected website, thus preventing from cyber attacks where an unknown listener is between the two legit endpoints. This kind of attack is called Man In The Middle (MITM). HTTPS is a mandatory standard for highly protected data and confidential online transactions like online banking and e-commerce. Its default port is the 443, instead of port 80 reserved to HTTP protocol. Let's see how the SSL/TSL protocols secure information more in detail.





    Security


    HTTP

    HyperText Tranfer Protocol is an application level protocol in the ISO/OSI stack, which includes a set of rules to send and receive text-based messages, allowing users to communicate data on the World Wide Web. It is not an encrypted protocol, so it is widely vulnerable to MITM attacks and interception. HTTP transfer the text, which the browser interprets and transforms in hypertext, and its default port is the number 80.

    SSL

    Acronym of Secure Socket Layer, is a secure protocol with the purpose of sending information securely over the internet. It is known websites uses it for creating a secure session area in order to host sensible information, such as login pages or online checkouts. The purpose of SSL is to avoid eavesdropping on the trasmission, so another person not included in the conversation can't know what you are communicating with a secured website. Only the two authenticated users by the protocol can recognize the data, all the others will see just meaningless encrypted trasmissions. Exempli gratia, SSL keeps your credentials and bank details between you and the authorized server, making possible and practical to do online business, as e-commerce websites. Moreover, all servers started to use an SSL certificate given by a Certificate Authority (CA), to validate their authenticity within the internet.

    TLS

    Acronym of Transport Layer Security, is the protocol providing authentication, privacy and data integrity between the two communicating endpoints. Is the most widely spread and used protocol nowadays, due to its evolution from the SSL protocol, which has been largely suppresed, although the terminology SSL/TLS openly says that it could possibly be used one or the other, making them interoperable. Also, the vastly fields of usage TLS has, makes it better to use compared to SSL. A couple of examples by just thinking about securely data exchanging by web applications are in the following paragraph. Key differences that make TLS more secure and efficient as a protocol are: message authentication, key material generation and the supported cipher suite containing newer and safer algorithms.

    Deployment


    Where is it used?

    Nowadays, HTTPS is almost ubiquitous. Most people don't really realize that in the URL address of mostly all pages visited there is a recurrent logo. The always present lock logo that appears in every browser has an important meaning. Not only it reassures us about the privacy our data has in that website, but tells us that we are connected to the server with the HTTPS protocol and a valid SSL certificate. Then, people could argue that surfing the web is not sufficient to define a protocol ubiquitous. Moreover, TLS is present in each web browsing session, Voice over IP (VOIP) as WhatsApp calls, remote desktop sessions like TeamViewer, file transfering, Virtual Private Network (VPN) connections, email, instant messaging ...
    Real examples contain big company names like Skype to keep conversations private, Google to keep its software suite trasmission secure (Drive, Calendar, Pay, ...) and many more.
    Thanks to HTTPS, our Personal Info (PI) and Sensitive Personal Info (SPI) is obscured thought the internet.

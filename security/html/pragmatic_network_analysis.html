    Pragmatic Network Traffic Analysis

    NTA

    Acronym for Network Traffic Analysis, its a branch of Computer Science focused in Computer Networking. It consist of methods for monitoring the network, in order to identify anomalies, such as security and operational issues. It is widely used in companies to:

    • Collect real-time and chronological record of data transferred over the network
    • Detect malware and computer viruses that operate over the network to spread themselves
    • Detect protocols and ciphers classified as highly vulnerable
    • Troubleshoot network issues
    • Enhanced network topology and delete blind spots

    The implementation of a solution which continuously monitor the network traffic helps to spot where to optimize the network performance, minimize the attack surface and enhance the security overall. Hereby, a couple of the best known tools for network analysis are describeb briefly.





    NMAP


    History

    The NMAP project is an open-source software distributed by Insecure.org. It's a port scanning tool which purpose is to spot available ports in a target machine(s) given an IP address or range of IP addresses; in order to identify which services are running over the network. It is widely used in cyber security by network administrators and crackers. The former use it to identify possible threads over the network, e.g. unauthorized server applications. The latter for analyse their target to collect as many information as possible. This features make nmap a must tool for pentesting.
    Fun Fact: in the movie Matrix Reloaded Trinity use Nmap to get into the electric central system, by bruteforce the SSH service and the bug CRC32 discovered in 2001.

    Usage

    Nmap is available for all Operating System platforms: Windows, Mac, GNU/Linux are the most important. It can be easily donwloaded from the Insecure.org webpage. Linux distros keep also a copy of the latest supported release in their own managed repositories, as an example, in Debian based distributions, it's can be installed by just digit the following commands on the terminal:
    $ sudo apt install nmap;

    • In order to perform a ping scan, hence scan the list of devices up and running on a given subnet:
      $ nmap -sp 192.168.1.1/24
    • To perform a version scanning for finding application versions:
      $ nmap -sV ip_address
    • To perform OS scanning:
      $ nmap -sV ip_address
    • To perform an aggressive scan, which includes OS detection, version detection, script scanning and traceroute:
      $ nmap -A ip_address
    Further information about how to use Nmap is available at nmap.org and in its man page on the terminal.

    Wireshark


    History

    Wireshark is an open-source software distributed by wireshark.org. It's a packet sniffer used to analyse standard network packet protocols. It was previously named Ethereal, its functionalities are similar to a tcpdump and incorporates a GUI to interact with. It is capable to recognize the structure of several network protocols, encapsulation, packet fields and is also able to reveal the content (if not encrypted). It's founder Gerald Combs needed a tool to capture and analyse network packets in 1998, then the software became popular and supported from global organization of network specialists and software developers. Government agencies, corporations, non-profits, and educational institutions use Wireshark for troubleshooting and teaching purposes.

    Usage

    Wireshark is available for all Operating System platforms: Windows, Mac, GNU/Linux as nmap. It can be easily donwloaded from the wireshark.org webpage. Linux distributions have the software available in their repositories. The principal features of Wireshark are:

    • Capturing data packets
    • Analyzing data packets
    • Filtering based on protocols, ip addresses, content, headers, byte lenght, ...
    • Metric and statistic about the network packets dump
    • Wireshark command line (CLI) to automate scripts without the GUI

    Further information about how to use efficiently Wireshark is available at the official documentation website.

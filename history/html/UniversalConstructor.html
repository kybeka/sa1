    1966 - Theory of Self-Reproducing Automata

    Universal Constructor

    John von Neumann's universal constructor is a self-replicating machine, designed in the 1940s without the use of a computer. The foundamental details of the machine were included in the "Theory of self-reproducing automata" and became the basis for the automata theory, the complex systems and the artificial life. Von Neumann coined the term "cellular automaton" to describe his machine in greater detail. He employed a two-dimensional grid of cells in which each cell can be in one of 29 states at any given time. At each cell updates its state at each timestep in response to the previous timestep's states of the surrounding cells.

    Von Neumann's goal was to design a machine whose complexity could grow automatically akin to biological organisms under natural selection.

    The Machine

    The machine was based on three separate components:

    • A memory tape with a description of the machine to be created. The memory contains a description of the universal constructor itself in the special case of self-replication.
    • The constructor itself, a machine capable of reading and interpreting the contents of the memory tape.
    • A constructing arm, directed by the constructor, used to build the machine described in the memory tape


    Following the usage of the universal constructor to establish a new machine encoded in the description, the copy machine is used to create a copy of that description, which is then handed on to the new machine, resulting in a working replica of the original machine that can continue to reproduce.
